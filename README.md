# Yosoro 模型游戏玩法指南

## 🎮 游戏简介

Yosoro 是一款基于 3D 模型的互动游戏，让你可以与各种虚拟角色进行实时交互。通过丰富的动作系统、表情系统和聊天功能，你可以与角色建立深厚的情感联系，打造属于自己的虚拟世界。

## 🚀 快速开始

### 1. 启动游戏

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

然后在浏览器中访问 `http://localhost:3000` 即可开始游戏。

### 2. 选择角色

首次进入游戏时，系统会默认加载 Bronya 角色。你可以通过顶部的 **模型选择** 面板切换不同的角色，游戏提供了超过 80 个不同风格的 3D 模型供你选择。

### 3. 选择场景

通过 **场景选择** 面板，你可以切换不同的背景环境，包括：
- 温馨卧室
- 现代客厅
- 阳光花园
- 午后咖啡
- 城市夜景
- 多种纯色背景

## 🎯 核心玩法

### 1. 角色交互

#### 鼠标交互
- **点击**：与角色进行基本互动
- **抚摸**：在角色身上移动鼠标，增加亲密度
- **拖拽**：移动角色位置

#### 动作控制
通过 **动作按钮** 面板，你可以让角色执行各种动作：
- **基础动作**：点头、摇头、跳跃、挥手、舞蹈
- **表情动作**：笑、哭、思考、惊讶
- **互动动作**：鞠躬、拍手、挥手
- **场景动作**：坐下、躺下、行走、奔跑

#### 表情控制
通过 **表情面板**，你可以调整角色的情绪状态：
- 开心、生气、悲伤、惊讶、害羞
- 放松、严肃、困惑、困倦

### 2. 聊天系统

点击 **聊天界面**，你可以与角色进行实时对话：
- 输入文字与角色交流
- 角色会根据你的对话内容做出相应的表情和动作
- 对话会增加亲密度，完成日常任务

### 3. 场景布置

通过 **家具系统**，你可以在场景中添加和摆放各种家具：
- 选择家具类型和样式
- 拖拽调整家具位置
- 旋转和缩放家具
- 为不同场景创建独特的布局

### 4. 任务系统

游戏提供了日常任务，完成任务可以获得奖励：
- **聊天任务**：与角色对话指定次数
- **互动任务**：与角色进行互动指定次数
- **抚摸任务**：抚摸角色指定次数

完成任务可以解锁新动作、增加亲密度，甚至解锁特殊成就。

### 5. 桌宠化功能

游戏支持桌宠化模式，让角色可以在你的桌面上自由活动：
- 角色会自主移动和做出表情
- 点击角色会有互动反馈
- 支持多角色同时存在

### 6. 记忆系统

随着与角色的互动增加，你会解锁各种记忆：
- 记录你与角色的重要时刻
- 展示角色的成长历程
- 提供独特的互动体验

## 🌟 特色功能

### 1. 智能 AI 系统

- **情感识别**：根据对话内容识别你的情绪
- **智能回应**：根据角色性格生成独特的回应
- **行为学习**：根据你的互动习惯调整角色行为

### 2. 高级物理引擎

- **真实物理碰撞**：角色与家具的互动更加真实
- **流畅动画**：动作过渡自然流畅
- **粒子效果**：互动时产生各种特效

### 3. 多角色系统

- **同时加载多个角色**：最多支持 3 个角色同时存在
- **角色互动**：不同角色之间可以进行互动
- **团队动作**：多个角色可以一起执行动作

### 4. 截图系统

- **拍照模式**：隐藏界面元素，专注于角色
- **自定义角度**：调整相机视角和位置
- **分享功能**：保存精彩瞬间并分享

### 5. 性能优化

游戏内置了先进的性能优化系统：
- **模型LOD**：根据距离自动调整模型细节
- **纹理压缩**：减少内存占用
- **垃圾回收**：自动清理不需要的资源
- **性能监控**：实时显示系统性能

## 📊 游戏数据

### 亲密度系统

通过与角色互动，你可以提升亲密度：
- **亲密度等级**：从 0 到 100
- **提升方式**：聊天、抚摸、完成任务
- **等级效果**：解锁新动作、新表情、新对话内容

### 成就系统

游戏包含多种成就，等待你去解锁：
- **初次见面**：完成第一次对话
- **好朋友**：亲密度达到 100
- **互动大师**：完成 100 次互动
- **聊天达人**：完成 100 次对话

### 统计数据

游戏会记录你的各种互动数据：
- 总聊天次数
- 总抚摸次数
- 总互动次数
- 连续登录天数

## ⚙️ 系统设置

### 1. 音频设置
- **语音开关**：开启/关闭角色语音
- **BGM开关**：开启/关闭背景音乐

### 2. 画面设置
- **画质等级**：从低到高多种画质选项
- **表情夸张度**：调整角色表情的夸张程度

### 3. 其他设置
- **主题切换**：浅色/深色/粉色主题
- **内存管理**：手动清理内存
- **API密钥**：设置AI对话API密钥
